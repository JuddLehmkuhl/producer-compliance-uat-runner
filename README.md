# Producer Compliance — UAT Runner

An interactive, self-guiding UAT test runner for the Producer Compliance (NIPR) feature.
Open the live app, work through the tests, and use **Export results** to share your feedback.

**▶ Live app:** https://juddlehmkuhl.github.io/producer-compliance-uat-runner/

Self-contained single page. Your progress auto-saves to your own browser (nothing is sent
anywhere); export a Markdown/JSON report when you're done. Built by PS Advisory.
